# compress/decompress

## Descomprimir

```code
tar -xvf file.tar

tar -xvzf dirb222.tar.gz

tar zxvf backups.tgz

tar xf tsetup.1.4.0.tar.xz

bzip2 -dk rockyou.txt.bz2

7z x archivo.7z

gunzip hdsentinel-017.gz
```

## Crear tar

```code
tar -cf Ypuffy.tar Ypuffy
```

## Ver archivos de un tar (no descomprime)

```code
tar -tvf Ypuffy.tar
```
